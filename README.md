# 🎵 Sonivo — Music Video Generator for Instagram

Web app to generate vinyl-style vertical music videos (1080×1080) from your audio files.

**Quick start:** Python 3.11+, FFmpeg, then `pip install -r requirements.txt` and `./start.sh` → [http://localhost:8000](http://localhost:8000)
